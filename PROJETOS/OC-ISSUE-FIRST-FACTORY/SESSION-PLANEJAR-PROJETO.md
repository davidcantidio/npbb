# SESSION-PLANEJAR-PROJETO - Planejamento de Projeto em Sessao de Chat

## Objetivo

Planejar o projeto novo a partir do PRD gerado e do bootstrap inicial da fase F1-FUNDACAO.

## Sessao Canonica

Leia e siga integralmente:

- `PROJETOS/COMUM/SESSION-PLANEJAR-PROJETO.md`

## Parametros Preenchidos

```text
PROJETO:       OC-ISSUE-FIRST-FACTORY
PRD_PATH:      PROJETOS/OC-ISSUE-FIRST-FACTORY/PRD-OC-ISSUE-FIRST-FACTORY.md
ESCOPO:        projeto completo
PROFUNDIDADE:  completo
TASK_MODE:     required
OBSERVACOES:   nenhuma
```

## Regra Local Adicional

- use o PRD gerado pelo scaffold como ponto de partida
- a fase inicial do projeto e `F1-FUNDACAO`
- em caso de conflito, `PROJETOS/COMUM/SESSION-PLANEJAR-PROJETO.md` prevalece
