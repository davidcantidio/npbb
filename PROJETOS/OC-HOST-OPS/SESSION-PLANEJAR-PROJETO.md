# SESSION-PLANEJAR-PROJETO - Planejamento de Projeto em Sessao de Chat

## Objetivo

Planejar o backlog real do host-side a partir do PRD atual e do worktree reconciliado.

## Sessao Canonica

Leia e siga integralmente:

- `PROJETOS/COMUM/SESSION-PLANEJAR-PROJETO.md`

## Parametros Preenchidos

```text
PROJETO:       OC-HOST-OPS
PRD_PATH:      PROJETOS/OC-HOST-OPS/PRD-OC-HOST-OPS.md
ESCOPO:        projeto completo
PROFUNDIDADE:  completo
TASK_MODE:     required
OBSERVACOES:   proxima unidade pratica do projeto; nao regredir para a issue bootstrap do scaffold
```

## Regra Local Adicional

- use o PRD real do host-side como fonte primaria
- o bootstrap F1 ja foi reconciliado; a fila atual do projeto e planning, nao execucao do scaffold
- em caso de conflito, `PROJETOS/COMUM/SESSION-PLANEJAR-PROJETO.md` prevalece
