# SESSION-PLANEJAR-PROJETO - Planejamento de Projeto em Sessao de Chat

## Objetivo

Planejar o backlog real da vertical Trading a partir do PRD atual e do worktree reconciliado.

## Sessao Canonica

Leia e siga integralmente:

- `PROJETOS/COMUM/SESSION-PLANEJAR-PROJETO.md`

## Parametros Preenchidos

```text
PROJETO:       OC-TRADING
PRD_PATH:      PROJETOS/OC-TRADING/PRD-OC-TRADING.md
ESCOPO:        projeto completo
PROFUNDIDADE:  completo
TASK_MODE:     required
OBSERVACOES:   proxima unidade pratica da vertical; nao regredir para a issue bootstrap do scaffold
```

## Regra Local Adicional

- use o PRD real da vertical como fonte primaria
- a fila atual do projeto e planning, nao execucao do scaffold
- em caso de conflito, `PROJETOS/COMUM/SESSION-PLANEJAR-PROJETO.md` prevalece
