# SESSION-PLANEJAR-PROJETO - Planejamento de Projeto em Sessao de Chat

## Objetivo

Replanejar fases futuras ou follow-ups do Mission Control a partir do PRD real e do estado atual da F2.

## Sessao Canonica

Leia e siga integralmente:

- `PROJETOS/COMUM/SESSION-PLANEJAR-PROJETO.md`

## Parametros Preenchidos

```text
PROJETO:       OC-MISSION-CONTROL
PRD_PATH:      PROJETOS/OC-MISSION-CONTROL/PRD-OC-MISSION-CONTROL.md
ESCOPO:        projeto completo
PROFUNDIDADE:  completo
TASK_MODE:     required
OBSERVACOES:   usar este wrapper para novas fases ou follow-ups depois da auditoria de F2; nao regredir para o scaffold da F1
```

## Regra Local Adicional

- o PRD atual do projeto ja representa backlog real do Mission Control; nao trate a F1 como ponto de partida do planejamento
- use este wrapper para abrir a proxima fase ou follow-ups estruturais depois do gate da F2
- em caso de conflito, `PROJETOS/COMUM/SESSION-PLANEJAR-PROJETO.md` prevalece
