# SESSION-CRIAR-PRD - Criacao de PRD em Sessao de Chat

## Objetivo

Gerar o PRD inicial a partir do intake do projeto e do scaffold existente.

## Sessao Canonica

Leia e siga integralmente:

- `PROJETOS/COMUM/SESSION-CRIAR-PRD.md`

## Parametros Preenchidos

```text
PROJETO:      OC-ISSUE-FIRST-FACTORY
INTAKE_PATH:  PROJETOS/OC-ISSUE-FIRST-FACTORY/INTAKE-OC-ISSUE-FIRST-FACTORY.md
OBSERVACOES:  nenhuma
```

## Regra Local Adicional

Use o intake gerado pelo scaffold como entrada inicial do PRD.
