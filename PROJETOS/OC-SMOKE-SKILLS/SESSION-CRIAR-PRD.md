# SESSION-CRIAR-PRD - Criacao de PRD em Sessao de Chat

## Objetivo

Gerar ou revisar o PRD do projeto-canario a partir do intake real e do guia de smoke.

## Sessao Canonica

Leia e siga integralmente:

- `PROJETOS/COMUM/SESSION-CRIAR-PRD.md`

## Parametros Preenchidos

```text
PROJETO:      OC-SMOKE-SKILLS
INTAKE_PATH:  PROJETOS/OC-SMOKE-SKILLS/INTAKE-OC-SMOKE-SKILLS.md
OBSERVACOES:  o PRD deste projeto descreve um canario de framework, nao backlog de produto; usar GUIA-TESTE-SKILLS.md como referencia de aceite
```

## Regra Local Adicional

Nao volte ao placeholder de scaffold. O PRD precisa permanecer alinhado ao `GUIA-TESTE-SKILLS.md`.
