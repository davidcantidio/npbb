# SESSION-CRIAR-INTAKE - Criacao de Intake em Sessao de Chat

## Objetivo

Gerar ou revisar o intake do projeto-canario a partir do `GUIA-TESTE-SKILLS.md`.

## Sessao Canonica

Leia e siga integralmente:

- `PROJETOS/COMUM/SESSION-CRIAR-INTAKE.md`

## Parametros Preenchidos

```text
PROJETO:       OC-SMOKE-SKILLS
INTAKE_KIND:   new-capability
SOURCE_MODE:   original
ORIGEM:        projeto-canario de framework descrito em GUIA-TESTE-SKILLS.md
ORIGIN_AUDIT:  nao_aplicavel
CONTEXT:       Projeto canario usado para validar roteamento, autonomia, execucao, revisao e auditoria da suite openclaw-* sem backlog de produto
OBSERVACOES:   usar GUIA-TESTE-SKILLS.md como contrato funcional do intake; nao tratar o projeto como scaffold generico
```

## Regra Local Adicional

Use este wrapper quando o canario precisar ser realinhado ao guia atual ou quando uma regressao do framework exigir remediacao documental.
